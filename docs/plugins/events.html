Lifecycle Events

Register with hook!(host, PluginEvent::Variant, callback). Hooks run in plugin load order. Return HookResult::cancel() to stop remaining handlers for that event.

EventSummaryWhenPayloadCancelable
config.loadedFires after the daemon config file has been parsed.During plugin directory scan, before any `.so` files are loaded.Raw YAML bytes of the active config file.yes
plugin.loadedFires after each plugin finishes initialization.Immediately after a plugin's `init` returns success.UTF-8 plugin name bytes.yes
daemon.startingFires before the HTTP server is constructed.After plugins are loaded and before Axum routes are wired.Empty.yes
daemon.startedFires when the HTTP server is listening.Right before `axum::serve` begins accepting connections.Listen address as UTF-8 bytes, e.g. `127.0.0.1:8080`.yes
daemon.stoppingFires during graceful shutdown.Before plugin `shutdown` functions are invoked.Empty.yes

Examples

config.loaded

Fires after the daemon config file has been parsed. During plugin directory scan, before any `.so` files are loaded.

Register

hook!(host, PluginEvent::ConfigLoaded, on_config_loaded);

Handler

extern "C" fn on_config_loaded(ctx: *const EventContext) -> HookResult {
    let ctx = unsafe { &*ctx };
    let payload = unsafe { std::slice::from_raw_parts(ctx.payload_ptr, ctx.payload_len) };
    let _yaml = std::str::from_utf8(payload).unwrap_or_default();
    HookResult::r#continue()
}

daemon.started

Fires when the HTTP server is listening. Right before `axum::serve` begins accepting connections.

Register

hook!(host, PluginEvent::DaemonStarted, on_daemon_started);

Handler

extern "C" fn on_daemon_started(ctx: *const EventContext) -> HookResult {

    HookResult::r#continue()
}
