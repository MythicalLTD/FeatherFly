cloudpanel.command_requested

A CloudPanel CLI command was requested.

When
Before CloudPanel command hooks run.
Payload
JSON: { operation, command, args, status?, duration_ms, error?, hook_handlers }.
Cancelable
No

Arguments are redacted before lifecycle payloads are emitted.

Use cases

  • Audit CloudPanel API usage
  • Track requested operations

Example

Registration and handler

Register

hook!(host, PluginEvent::CloudPanelCommandRequested, on_cloudpanel_requested);

Handler

extern "C" fn on_event(ctx: *const EventContext) -> HookResult {
    let ctx = unsafe { &*ctx };
    let payload = unsafe { std::slice::from_raw_parts(ctx.payload_ptr, ctx.payload_len) };
    let _json = std::str::from_utf8(payload).unwrap_or_default();
    HookResult::r#continue()
}
